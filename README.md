# Node - Shopping List Api

Node shopping list api using a json file as storage.

list requirements:

1. listName
2. category
3. quantity
4. img

GET /lists
GET /list/listId
POST /lists
PUT /list/listId
DELETE /list/listId

## Installation

Start project by installing dependencies

```bash
npm install
```

## Usage

```python
Run server

node --watch server
```
